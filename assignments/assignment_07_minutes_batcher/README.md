## Assignment 7: Minutes & Action Items Batcher

Turn messy meeting transcripts into concise minutes and action items, including
parallel processing of many transcripts at once.

### Concepts practiced
- Runnable.batch for parallel execution
- Prompt templates and clean formatting with `StrOutputParser`

### Your task
- Implement the bodies in `minutes_batcher.py` according to the docstrings.
- Provide both per-transcript and batched processing.

### Run
```
python minutes_batcher.py
```
